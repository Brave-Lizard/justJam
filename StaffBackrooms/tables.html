User Story Solution
User Story # 1 - Visual Novel Player that loves the story Playing through JustJam, I've been able to meet some characters throughout this ridiculous world. It's funny and engaging, but some characters have obviously been given more effort than others. There are multiple characters that I would like to interact with more, but the story ends without getting to do anything. Solution: Create in-depth interactions between characters.
User Story # 2 - RPG Player that likes customization When you start JustJam, you get to choose a name for your character, but that's it. The original look is always the same, whether a boy or a girl and a farmer or a time traveler. Obviously this needs some work, and the story should be more focused on the main character. Solution: Create character customization.
